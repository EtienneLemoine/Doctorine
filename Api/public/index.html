Rutes Examples

Sending information

When the information is sent to the database divided into different objects. Use the table name preceded by the word "info". For example: infoUser or infoMedic.

ID and ModelID

The ID is the primary key of the table. The ModelID is the ID of the model that the ID belongs to.

When you send a specific ID for a table, use the name of the table plus "ID". For example: UserID or MedicID.

All tables that include a foreign ID use the "ModelID" format. For example, when viewing a medic, the infoUser is displayed, the infoMedic is appended, and the infoMedic contains the ClinicID.
